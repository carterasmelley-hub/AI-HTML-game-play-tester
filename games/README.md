# Add local games here

Use either of these patterns:

- `games/my-game.html`
- `games/my-game/index.html`

Optional metadata files:

- `games/my-game.game.json` for `games/my-game.html`
- `games/my-game/game.json` for `games/my-game/index.html`

Metadata example:

```json
{
  "title": "Neon Knockout",
  "description": "Fast dodge game with glowing hazards.",
  "tags": ["Arcade", "Skill"],
  "thumbnail": "cover.png",
  "featured": true,
  "order": 1
}
```
